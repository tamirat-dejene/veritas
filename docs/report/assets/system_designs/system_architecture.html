Component Directory

Details on Gateway and Core domain dispatching paths:

GIN PROXY API Gateway

Processes token verification, tenancy injection, and route-level rate limiting using high-efficiency Go Gin middleware.

REDIS Leaky-Bucket Limiter

Synchronously queries Redis to evaluate tenant rate limits, request thresholds, and active session validation states.

DOMAINS Bounded Contexts

Discrete services (Auth, Enterprise, Exam, Candidate, Payments) manage their logic isolations independently with REST APIs.

Details on asynchronous system coordination using Kafka broker topics:

PUB-SUB Loose Coupling

Prevents cascading service failures. Core services publish mutations immediately then complete synchronous request cycles.

WORKER Notification Dispatcher

An event consumer dedicated to sending transaction and status updates to users, avoiding latency bottlenecks during exams.

ML GRADERS Asynchronous Scoring

Subscribed to ready_for_grading, the scoring service triggers heavy deep learning evaluations sequentially without locking databases.

Persistence architectures and system third-party API dependencies:

DB-PER-SVC Isolated Data Stores

Forces clean service decoupling. No database sharing allowed; references must cross-query APIs or depend on Kafka consistency.

DEEPFACE Visual Audits

Proctoring layer sends frame slices to the DeepFace endpoint for identity and gaze verification against reference registration images.

HUGGING FACE NLP Scoring Engines

Automates the parsing of short textual answers, computing deep semantic similarities against dynamic model guidelines.
